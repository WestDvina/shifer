# ShIFER

**Sh**ared **I**SO **F**ast **E**xchange **R**elay — мониторинг оригинальных ISO-образов Windows для РФ и РБ.

Ссылки собираются автоматически из Microsoft Q&A, где специалисты Microsoft публикуют прямые ссылки на скачивание. Каждая ссылка живёт ~24 часа — на сайте отображается обратный отсчёт.

## Как это работает

1. Парсер проверяет Microsoft Q&A каждые 15 минут
2. Находит новые ответы с прямыми ссылками на ISO
3. Ссылки отображаются на сайте с таймером обратного отсчёта

## Если ссылки нет

Нажмите «Запросить ISO» — сайт сгенерирует готовый текст для вопроса на Microsoft Q&A. Опубликуйте его — специалист Microsoft ответит со ссылкой.

## Разработка

```
├── docs/          # GitHub Pages (index.html, style.css, app.js)
├── parser/        # Парсер Microsoft Q&A
├── gather.py      # Сборщик данных
└── .github/       # GitHub Actions (запуск парсера каждые 15 мин)
```
