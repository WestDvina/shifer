Запросить ISO у Microsoft

Выберите версию Windows и разрядность — будет сгенерирован готовый текст для вопроса на Microsoft Q&A.

Готовый текст вопроса

Как опубликовать запрос:

  1. Нажмите кнопку ниже — откроется страница создания вопроса на Microsoft Q&A
  2. Войдите под своей учётной записью Microsoft
  3. Выберите тег: Windows для дома → Windows 10/11 → Установка и обновление
  4. В поле Заголовок напишите, например: Please provide a link to the Windows 10 ISO image
  5. В поле Текст вопроса вставьте скопированный текст (Ctrl+V или Cmd+V)
  6. Нажмите Опубликовать

Специалист Microsoft ответит в течение нескольких минут и предоставит прямую ссылку на ISO.
